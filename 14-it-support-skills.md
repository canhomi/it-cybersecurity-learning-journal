# 14 - IT Support Skills

## Objective

Document the main technical and professional skills required for entry-level IT Support, Help Desk and Service Desk roles.

## What is IT Support?

IT Support is the area responsible for helping users, devices and systems work correctly.

An IT Support professional helps solve technical problems related to:

- Computers
- Operating systems
- User accounts
- Passwords
- Networks
- Printers
- Applications
- Email
- Microsoft 365
- VPN
- Security basics

## Why IT Support Matters

IT Support is one of the most common entry points into the IT industry.

It builds strong foundations in:

- Troubleshooting
- User support
- Windows
- Networking
- Microsoft tools
- Documentation
- Security awareness
- Business environments

## Core Technical Skills

Important technical skills for IT Support include:

## Common Support Scenarios

### Password Reset

Possible checks:

```text
Confirm user identity
Check account status
Reset password according to policy
Verify MFA
Ask user to test login
Document the ticket
```

### No Internet Connection

Possible checks:

```text
Check Wi-Fi or cable
Check IP address
Ping 8.8.8.8
Ping google.com
Check DNS
Check VPN
Check firewall
Document findings
```

### Slow Computer

Possible checks:

```text
CPU usage
Memory usage
Disk usage
Startup apps
Updates
Malware scan
Available storage
Background processes
```

### Email Problem

Possible checks:

```text
Can the user log in?
Is the mailbox full?
Is MFA working?
Is the account locked?
Is Outlook connected?
Is the problem affecting one user or many users?
```

### Printer Problem

Possible checks:

```text
Is the printer powered on?
Is it connected to the network?
Is there paper?
Is there toner or ink?
Is the correct printer selected?
Is the printer queue stuck?
```

## Ticket Documentation

A good ticket should include:

```text
User affected
Device affected
Problem description
Time of issue
Error message
Steps tested
Solution applied
Final result
Next action if needed
```

## Example Ticket Note

```text
User reported no internet connection on Windows laptop.

Checked Wi-Fi connection and IP configuration.
Device had valid IP address.
Ping to 8.8.8.8 worked.
Ping to google.com failed.
DNS issue suspected.

Changed DNS configuration and tested again.
Internet access restored.

Ticket resolved.
```

## Cybersecurity Awareness

IT Support is often the first line of defense.

Support technicians should recognize:

- Phishing emails
- Suspicious links
- Malware symptoms
- Unusual login behavior
- MFA problems
- Account lockouts
- Unknown software
- Suspicious browser pop-ups
- Users sharing passwords

## Common Mistakes

- Not documenting actions
- Restarting without checking symptoms
- Ignoring user details
- Using technical language with non-technical users
- Not verifying if the problem was solved
- Closing tickets too early
- Ignoring possible security signs
- Not escalating when needed
