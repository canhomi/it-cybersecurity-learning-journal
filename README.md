This repository documents my practical learning journey in IT Support, Linux, Windows, Networking, Cloud, Microsoft 365 and Cybersecurity.

## Objective

Build a professional portfolio with hands-on labs, technical notes, commands, troubleshooting documentation and security analysis exercises.

## Current Focus

- IT fundamentals
- Linux command line
- Virtual machines
- Networking basics
- Windows and Ubuntu Server labs
- Wazuh and SOC fundamentals
- Defensive cybersecurity practices

## Practical Labs

### 01 - Linux Fundamentals
Basic Linux commands, file system navigation, permissions, processes and services.

### 02 - VirtualBox Lab
Creation and configuration of virtual machines for IT and cybersecurity training.

### 03 - Ubuntu Server Lab
Basic server installation, update, network verification and service management.

### 04 - Kali Linux Lab
Use of Kali Linux in a legal and controlled lab environment for learning security tools.

### 05 - Networking Basics
Practice with ping, traceroute, DNS lookup, IP configuration and basic network troubleshooting.

### 06 - Wazuh SOC Lab
Initial SOC lab using Wazuh to understand agents, logs, alerts and basic security monitoring.

### 07 - Windows Lab
Windows virtual machine setup for support, administration and security practice.
