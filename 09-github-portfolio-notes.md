# 09 - GitHub Portfolio Notes

## Objective

Document how I started using GitHub as a professional portfolio for IT Support and Cybersecurity learning.

## What is GitHub?

GitHub is a platform used to store, organize and share technical projects, documentation, code, scripts and learning notes.

For IT and cybersecurity, GitHub can be used as a public portfolio to show practical knowledge and technical progress.

## Why This Matters

A resume says what I studied.

A portfolio shows what I practiced, documented and understood.

GitHub helps demonstrate:

- Practical learning
- Technical documentation
- Lab experience
- Command-line knowledge
- Cybersecurity mindset
- Organization
- Continuous improvement

## What is a README?

A README file is the main introduction page of a repository.

It explains:

- What the project is
- Why it exists
- What tools are used
- What was learned
- How the project is organized

## What is a Commit?

A commit is a saved change in the repository history.

Each commit documents progress and shows that the project is being updated over time.

Example commit messages:

```text
Add Linux basics notes
Add VirtualBox lab notes
Add Wazuh SOC lab notes
Fix markdown code block formatting
```

## Why Markdown is Useful

Markdown is a simple formatting language used to write technical documentation.

Examples:

```markdown
# Main Title
## Section Title
- Bullet point
```bash
command here
```
```

Markdown is useful because it makes documentation clean, readable and professional.

## Common Mistakes

- Creating a GitHub account but not documenting anything
- Uploading random files without structure
- Writing unclear notes
- Not explaining the objective of each lab
- Not updating the portfolio regularly
- Uploading sensitive information
- Using screenshots with private data
- Publishing passwords, tokens or API keys

## Security Notes

Never publish:

- Passwords
- Private keys
- API tokens
- Personal documents
- Internal company data
- Private IPs with sensitive context
- Customer or client information
- Screenshots containing personal information
